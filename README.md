# ffcmd
Command line tool for Firefox.

# Setup

* Download zip file from this page.
* Unpack zip file 
* Drag addon/ffcmd.xpi and drop it to your firefox.
* Set PATH environment variable to bin/ffcmd

# Usage

* List tabs

`$ tab list`

* Show url of active tab

`$ tab current`

`$ tab open [url]`

`$ tab select [index]`

`$ tab set-url [url]`

`$ tab close`

`$ tab ready-state`
